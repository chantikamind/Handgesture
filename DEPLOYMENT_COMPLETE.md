## ğŸ”§ What Happens During Deployment

1. **GitHub Integration**
   - Vercel checks your GitHub repository
   - Monitors for new commits

2. **Build Phase**
   - Installs Python 3.11
   - Runs: `pip install -r api/requirements.txt`
   - Prepares your Flask app

3. **Deployment Phase**
   - Creates serverless functions
   - Routes all requests to Flask app
   - Sets up your domain
   - Activates SSL/HTTPS

4. **Activation**
   - Your app goes live
   - Gets unique URL: `https://your-project-name.vercel.app`
   - Available 24/7 with 99.95% uptime
